Example

Once the below button is clicked, it will redirect you to a VR webpage in which you will be presented with 3 different components. One of them is a GLTF model that I have implemented, these will be used in the final project and will be placed throughout the art gallery that I will be creating. The other 2 components are a traffic light model which was created that once the button is pressed, the lights will change from green to red which will then show the user that they can safely walk towards the art gallery (this was created as an extra and fun feature which I thought will give more interactivity to the user), and the last component is a rotating sphere which is there as an abstract art piece which is on display and infinitely rotates.
